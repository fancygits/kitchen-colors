# Textures

Drop the finish close-up photos here. The app converts each image to
grayscale, normalizes it to neutral brightness, then soft-light blends
it over the color-tinted cabinet region — preserving the kitchen photo's
own lighting while adding real grain/texture detail.

Any square JPEG or PNG works. The image is scaled to 512px on load and
tiled across the masked area.
